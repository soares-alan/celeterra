## 📂 Project Architecture

```bash
corporate-website/
├── index.html              # Main entry point
├── css/
│   ├── style.css           # Custom styles
│   └── bootstrap.min.css   # Bootstrap framework
├── js/
│   ├── main.js             # Core functionality
│   └── bootstrap.bundle.min.js

    ├── images/             # Optimized media
    ├── icons/              # SVG assets
    └── fonts/              # Custom fonts
```
